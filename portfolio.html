Travel Planner

Allows input from user a point of origin and destination. Routing is limited to travel by land only. Shows current weather for the destination and a 5-day forecast. Search history will be saved and can be retrieved from the drop-down list. Used momentjs, Bulma, server APIs (mapbox, ipdata, ipgeolocation, & openweathermap)

Work-Day-Scheduler

User can add the important events to a daily planner. So that he can manage his time effectively. Built with HTML, CSS, Javascript, MomentJS and Bootstrap.

Password Generator

For the Password Generator assignment we were asked to create a an application that would provide a random password to a user based on length, letter, number, and special character options that the user needed. The application uses HTML, CSS, Javascript.

Coding Quiz
